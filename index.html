About nuzi.ai

Revolutionizing how you collect and analyze user feedback

Simple Integration, Powerful Insights

Sentiment Analysis Powered by ML

Ridiculously Easy to Install

Add nuzi.ai to your app in under 5 minutes with our simple installation process.

install.sh 
# Install the nuzi.ai SDK
npm install @nuzi/voice-feedback

# Then add just two lines of code

import { NuziVoiceFeedback } from '@nuzi/voice-feedback';

// Initialize with your API key
NuziVoiceFeedback.init('YOUR_API_KEY');
